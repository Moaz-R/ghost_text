# GhostText 👻

this is a simple c++ program I made for the hack club #horizons track. 

basically, it takes a secret message and hides it inside normal text using zero-width characters (characters you cant see). 

## how it works

1. The program converts the hidden message into binary first.
2. After that, every bit becomes an invisible Unicode character.
3. Finally, those characters get appended to the end of the cover text.
